# Introduction

`np.Templating` is a template language plugin for NotePlan that lets you insert variables and method results into your notes.  It will also let you execute custom JavaScript constructed in the templates providing a rich note taking system.

## Configuration

`np.Templating` has a number of [settings](/docs/settings) which you can use to control how data is supplied and formatted when rendering templates.

:::info
For more information on `np.Templating` settings, visit [settings](/docs/settings)
:::

## Using Templating Commands
`np.Templating` provides a variety of built-in commands which can be invoked using NotePlan Plugin Command Bar (`Command-J`)

### Appending or Inserting Template

**Step 1:** Invoke NotePlan Command Bar (`Command-J`)

**Step 2:** Select `np:append` or `np:insert`

![Template Chooser](/img/template-chooser.png)

### Using Quick Notes
`np.Templating` also provides support for "Quick Notes" using the `np:qtn` command, which are more specific templates for creating new documents.

By default, Quick Notes templates located in "@Templates/üóí Quick Notes".  You can change the name of Quick Notes folder by defining the [Quick Notes setting](/docs/settings) value.

**Step 1:** Invoke NotePlan Command Bar (`Command-J`)

**Step 2:** Select `np:qtn`

**Step 3:** Select desired template

After Quick Note template is selected, a new note will be created in the defined location for selected template

![Template Chooser](/img/quicknotes-chooser.png)

### Quick Note Template Required Fields
Each Quick Note must contain the following fields in template header

- `title` - contains the template title (this is what will appear in `np:qtn` template chooser)

- `newNoteTitle` - will be name of new note.  You can use any `np.Templating` module method

- `folder` - path to folder where new note will be created

### Integrating into your plugins

```javascript
import NPTemplating from 'NPTemplating'

export async function templatingHelloWorld(): Promise<void> {
  try {
    const result = await NPTemplating.renderTemplate('Template - Hello World', {})

    Editor.insertTextAtCursor(result)
  } catch (error) {
    console.log('templatingHelloWorld', error)
  }
}
```
